# D-surface

A D-surface is a basic component used to contain any type of content. It renders a white div with some padding.

D-surface composes the D-box component and thus can be styled with the same style props

export const Template = args => ({
  components: { DSurface, DBox },
  template: `
  <DBox bg="grey.1" p="4">
      <DSurface mb="4">I am a surface</DSurface>
      <DSurface>I am another surface</DSurface>
  </DBox>`
});
